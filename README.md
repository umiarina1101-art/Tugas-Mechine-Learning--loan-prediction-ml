# Prediksi Kelayakan Pinjaman Nasabah Bank dengan Machine Learning

## Deskripsi Proyek
Proyek ini bertujuan untuk memprediksi kelayakan pinjaman nasabah bank menggunakan tiga algoritma Machine Learning: Logistic Regression, Support Vector Machine (SVM), dan Random Forest.

## Dataset
- Jumlah data awal: 381 nasabah
- Jumlah fitur: 12 fitur
- Target: Loan_Status (Layak / Tidak Layak)

## Hasil Terbaik
- **Model Terbaik:** Random Forest (default)
- **Akurasi:** 88,99%
- **Precision:** 85,00%
- **Recall:** 94,44%
- **F1-Score:** 89,47%
- **ROC-AUC:** 94,80%

## Fitur Paling Penting
1. **Credit_History** (47,96%)
2. ApplicantIncome (10,58%)
3. LoanAmount (8,85%)

## Cara Menjalankan
1. Buka Google Colab
2. Upload file `loan_prediction_final.ipynb`
3. Upload dataset `loan_data.csv` ke folder yang sesuai
4. Jalankan cell per cell dari atas ke bawah
